# WinSpeedBall 使用声明

版本：2026-07-16.1

适用产品版本：WinSpeedBall `3.7.0 Developer Beta`

WinSpeedBall 是用于视频学习、网页阅读、OCR、AI 总结与用户主动配置自动化的学习效率工具，不提供违法违规用途授权。

## 允许用途

- 对用户有权访问的内容进行视频控制、文字识别、学习笔记、翻译、总结和页面辅助。
- 在网站规则、课程纪律和适用法律允许的范围内使用用户脚本与自动化。

## 禁止用途

- 考试作弊、代答代交、伪造学习记录或绕过学校、平台及用人单位规则。
- 未授权访问、破解、绕过付费或访问控制、恶意自动化、欺诈、骚扰或监控他人。
- 侵犯版权、隐私、个人信息权益或其他合法权益。
- 导入恶意脚本，或处理用户无权处理的个人信息、机密内容和受保护资料。

## 数据与第三方服务

本地账户、设置、日志、截图、OCR 结果、网页语音转写文字以及按 AI 服务保存的问题草稿、回复和历史默认保存在当前浏览器。只有用户主动发送或开启自动发送时，相关文字才会传输到用户选择的 AI 服务。第三方服务适用其自身条款与隐私规则。

网页语音功能只在用户主动开始后获取当前 Edge 标签页播放的声音，不读取麦克风。原始录音只在内存中用于本地 Whisper 转写，识别完成或取消后释放，不保存为文件，也不上传到项目服务器。

Developer Mode 的 SDK 脚本在受限沙箱中运行，只能调用用户明确确认的 `WSB` 能力。SDK 草稿、隔离存储和授权保存在本地；短期令牌、页面确认和会话只保存在浏览器会话存储中。脚本调用 `ai.request` 时，相关文本会发送到用户选择的 AI 服务，但 API Key 不会开放给脚本。

`3.7.0` 推荐使用更短的 WSB 接口名，旧接口名仅作为兼容别名继续保留；接口改名不会扩大脚本权限或数据发送范围。
