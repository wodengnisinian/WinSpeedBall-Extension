# 更新日志

## 3.7.0 Developer Beta - 2026-07-17

### Developer Mode

- SDK Runtime 和授权绑定版本升级为 `3.7.0-beta`，旧版本授权需要重新确认。
- 编辑器新增行数、字符数、能力数量和保存状态实时统计。
- 新增 SDK 草稿复制功能，副本使用独立标识并自动更新脚本名称。
- 新增 `Ctrl+S` 快速保存当前 SDK 草稿。
- SDK API 契约测试新增所需能力实时提示。
- 对外接口采用更短的推荐名称：`video.all/status/rate/volume` 和 `book.status`；旧 `getAll/getStatus/setRate/setVolume` 名称继续兼容。
- 长脚本标识使用稳定哈希后缀生成注册名和隔离环境名，避免截断后发生冲突。
- 高级 SDK 脚本的单脚本隔离存储上限由 256 KiB 提升至 5 MiB；单值 64 KiB 和 100 键限制保持不变。
- `WSB.video.status()` 补齐媒体数量、总时长、已播放、播放状态、自动播放和倍速锁等公开字段。
- 新增 `qa.read`、`ai.read` 只读能力及 `qa.latest/ocr/voice`、`ai.latest/history` 公开接口，脚本可在明确授权后读取问题与 AI 答案。
- 新增只读能力 `book.read` 和公开接口 `WSB.book.status()`，脚本可在绑定当前网页后读取图书当前选项、封底状态与检测倒计时。
- 保留最多 20 个草稿、导入、导出、校验、安全删除、授权会话和真实 API 调用。

### AI 与问题获取

- AI 提问支持 DeepSeek、OpenAI、Claude 和本地模型的独立工作区及历史记录。
- AI 服务按钮统一使用 DS、OAI、CLD 和 LM 简称，并排显示在一行。
- 未配置的 AI 服务会显示明确提示，并可直接进入对应设置。
- 合并框选截图与 OCR 结果，新增 Edge 网页声音和本地 Whisper 转写。
- OCR、网页语音与 AI 回复统一执行简体中文和英文正规化。
- AI 回复使用固定独立窗口，新回复会完整关闭旧回复窗口。
- AI 响应采用限量流式读取，超过 2 MiB 时立即取消，避免异常响应持续占用内存。
- 使用声明同步新增网页语音本地处理说明，声明版本更新为 `2026-07-16.1`。

### 图书功能

- 新增学习通课程章节内嵌图书支持，适配 `mooc1.chaoxing.com/mycourse/studentstudy` 页面。
- 精确识别 `ans-insertbook-module`、`module="insertbook"` 和 SSLibrary EPUB 阅读器。
- 图书控制改为先扫描全部 iframe，再锁定评分最高的阅读器执行单次翻页。
- 新增“检测图书”按钮，并显示阅读器类型、框架、页码和实际翻页方式。
- 自动避开学习通课程正文的章节导航，防止误点“上一节”或“下一节”。
- 手动与自动翻页会按需申请图书 iframe 来源权限，并记录完整执行日志。
- 自动补充 `resapi.chaoxing.com` 与 `epub.sslibrary.com` 两级阅读器权限。
- 图书阅读器新增 MAIN 主环境原生控制核心，使用浏览器原生点击、页面控制器和方向键兜底。
- 新增旧版超星 JPath/Readweb 图片阅读器适配，直接调用 `readweb.nextPage` 和 `readweb.prevPage` 原生接口。
- 自动补充 `readsvr` 系列旧版图书阅读器框架权限。
- 图书界面拆分为“图书自动翻阅”和“图片自动翻阅”，分别控制阅读器页码与图片序列。
- JPath MAIN 控制核心升级为 V7，使用真实 frame 列表定位重定向后的阅读器，支持动态子框架并排除学习通外层图书壳。
- 学习通版本自动翻阅间隔最低支持 2 秒，并使用快速定时器配合 30 秒后台唤醒保护。
- 学习通版本按 400、300、250、150、50 秒的递减间隔检查 `#pagejump` 当前选项；检测到“封底页”后自动停止，最后以 50 秒间隔持续检查。
- 学习通界面新增封底页检测状态区，实时显示检测状态、当前选项和下次检测倒计时。
- 移除无变化方向键的成功判定，避免界面显示成功但图书没有翻页。

### 工程结构

- 按后台、界面、网页注入、OCR、语音、SDK、工作区和资源重新整理目录。
- 项目根目录只保留 Manifest、更新日志、说明、隐私政策和 Git 配置。
- 同步更新 Manifest、动态脚本注入、离屏页面、可信页面和测试路径。
- README、完整使用说明和插件内更新日志同步到 `3.7.0`。
- 新增项目结构守卫，验证入口、HTML 依赖、后台依赖和资源路径。
- 媒体主环境扫描合并为单次 DOM 遍历，减少动态页面中的重复查询。
